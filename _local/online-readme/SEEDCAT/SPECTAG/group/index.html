group

Group catalogs are split into two independent footprint region, NGC or SGC.

Each {FOOTPRINT} (NGC or SGC) generates multiple files:

  • {FOOTPRINT}_galaxy: summary of galaxy catalog

  • {FOOTPRINT}_group: summary of group catalog

  • {FOOTPRINT}_1: membership galaxy

  • {FOOTPRINT}_2: membership group

{FOOTPRINT}_galaxy

Summary:

group catalog

Naming Convention:

{FOOTPRINT}_galaxy.txt, where {FOOTPRINT} is the footprint region NGC or SGC.

File Type:

ASCII

{FOOTPRINT}_group

Summary:

group catalog

Naming Convention:

{FOOTPRINT}_group.txt, where {FOOTPRINT} is the footprint region NGC or SGC.

File Type:

ASCII
